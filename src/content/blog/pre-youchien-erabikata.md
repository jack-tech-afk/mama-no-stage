「来年の入園に向けて、プレ幼稚園に通わせたほうがいいのかな」「そもそもプレ幼稚園って何をするの？」と疑問に思っているパパ・ママへ。この記事では、プレ幼稚園の基本から選び方、見学時のチェックポイント、費用、さらには幼稚園・保育園・認定こども園の違いまで、入園前に知っておきたい情報をまとめました。

## プレ幼稚園とは

### プレ幼稚園の概要

プレ幼稚園（プレ保育）とは、正式な入園の前に幼稚園の雰囲気を体験できるプログラムのことです。多くの幼稚園が2歳児（満3歳になる年度の子ども）を対象に実施しています。

### プレ幼稚園の目的

- 子どもが集団生活に慣れる
- 親が幼稚園の方針や雰囲気を確認できる
- 入園後のスムーズな適応を目指す
- 幼稚園側も子どもや保護者を知る機会になる

### プレ幼稚園の形態

| 形態 | 頻度 | 時間 | 特徴 |
|------|------|------|------|
| 週1回型 | 週1回 | 2〜3時間 | 最も一般的。曜日固定が多い |
| 月1〜2回型 | 月1〜2回 | 1〜2時間 | お試し的に参加できる |
| 毎日型（満3歳児クラス） | 週5日 | 午前中〜14時頃 | 正規の入園に近い形 |
| 親子参加型 | 週1回程度 | 1〜2時間 | 親も一緒に参加 |
| 子どものみ参加型 | 週1〜2回 | 2〜3時間 | 母子分離の練習にもなる |

## プレ幼稚園の探し方

### 情報収集の方法

#### 1. 自治体の子育て支援窓口に問い合わせる

市区町村の子育て支援課や子育てひろばで、地域のプレ幼稚園情報を入手できます。一覧表を配布している自治体もあります。

#### 2. 幼稚園の公式サイトをチェック

気になる幼稚園のホームページには、プレ保育の案内が掲載されていることが多いです。秋〜冬にかけて次年度の募集情報がアップされます。

#### 3. 近所のママ友・子育てひろばで情報交換

実際に通っているママからの口コミは貴重な情報源です。子育て支援センターや公園での交流で情報を集めましょう。

#### 4. 幼稚園フェア・合同説明会に参加

一部の地域では、複数の幼稚園が集まる合同説明会が開催されます。効率的に情報収集できるので、開催情報をチェックしましょう。

### 申し込み時期

| 時期 | やること |
|------|---------|
| 9〜10月 | 各幼稚園のプレ保育の情報が出始める |
| 10〜11月 | 説明会・見学会が開催される |
| 11〜12月 | 申し込み受付開始（先着順が多い） |
| 1〜2月 | 二次募集がある場合も |
| 4月 | プレ保育スタート |

人気の幼稚園は申し込み初日に定員に達することもあります。情報収集は早めに始めましょう。

## 見学時のチェックリスト

幼稚園見学は、パンフレットだけでは分からない雰囲気を肌で感じる大切な機会です。

### 園の環境

- 園舎は清潔で安全か（角の処理、柵の高さ）
- 園庭の広さと遊具の状態
- 教室の明るさ・温度管理
- トイレは子どもサイズか、清潔か
- 送迎時の駐車場・駐輪場の有無

### 先生・保育内容

- 先生の表情は明るいか、子どもへの接し方は丁寧か
- 子どもたちは楽しそうに過ごしているか
- カリキュラムの内容（自由遊び中心か、一斉活動中心か）
- 英語・体操・音楽など課外活動の有無
- 給食の有無と内容（アレルギー対応の可否）

### 運営面

- 預かり保育の有無と時間（共働きの場合は重要）
- バス通園の有無とルート
- 行事の頻度と保護者の参加負担
- 保護者会・PTAの活動内容
- 連絡方法（アプリ・連絡帳・紙の手紙）

### 確認すべき質問リスト

見学時に聞いておきたい質問をまとめました。

| カテゴリ | 質問例 |
|---------|--------|
| 費用 | 入園料・月額保育料・制服代・教材費の総額は？ |
| 送迎 | バスの運行時間・ルートは？雨の日の対応は？ |
| 給食 | 完全給食？お弁当の日は何日？アレルギー対応は？ |
| 預かり | 預かり保育は何時まで？長期休暇中もある？ |
| 教育方針 | のびのび系？お勉強系？モンテッソーリ？ |
| 安全 | 防犯対策は？災害時の避難計画は？ |

## 費用の目安

### プレ幼稚園の費用

| 費用項目 | 金額の目安 |
|---------|-----------|
| 入会金 | ¥5,000〜¥30,000 |
| 月謝 | ¥3,000〜¥10,000 |
| 教材費 | ¥2,000〜¥5,000（年間） |
| 保険料 | ¥1,000〜¥2,000（年間） |

### 正式入園時の費用

| 費用項目 | 金額の目安 |
|---------|-----------|
| 入園料 | ¥30,000〜¥150,000 |
| 月額保育料 | 無償化により0円（上限¥25,700） |
| 制服・体操服 | ¥20,000〜¥50,000 |
| 用品一式 | ¥10,000〜¥20,000 |
| バス代 | ¥3,000〜¥5,000/月 |
| 給食費 | ¥3,000〜¥6,000/月 |

2019年10月から始まった幼児教育・保育の無償化により、3〜5歳の保育料は基本的に無料です。ただし、給食費・バス代・制服代などは実費負担となります。

## 幼稚園 vs 保育園 vs 認定こども園の違い

「そもそも幼稚園と保育園、認定こども園って何が違うの？」という方のために、違いを整理します。

| 項目 | 幼稚園 | 保育園（認可） | 認定こども園 |
|------|--------|-------------|-------------|
| 管轄 | 文部科学省 | 厚生労働省 | 内閣府 |
| 対象年齢 | 3〜5歳 | 0〜5歳 | 0〜5歳 |
| 保育時間 | 4時間（標準） | 8〜11時間 | 施設により異なる |
| 入園条件 | 特になし | 保育の必要性あり（共働き等） | 施設により異なる |
| 目的 | 教育 | 保育 | 教育＋保育 |
| 給食 | 任意（弁当の園も多い） | 義務 | 義務 |
| 長期休暇 | 夏休み・冬休み・春休みあり | 基本なし | 施設により異なる |

### どれを選ぶべき？

- **専業主婦（夫）家庭** → 幼稚園または認定こども園
- **共働き家庭** → 保育園または認定こども園
- **パートタイム勤務** → 預かり保育が充実した幼稚園 or 認定こども園

近年は共働き家庭の増加に伴い、預かり保育を充実させる幼稚園も増えています。「幼稚園＝専業主婦向け」とは限らなくなってきています。

### お名前付けの効率化

入園準備で最も大変なのが「お名前付け」です。効率的に進めるコツを紹介します。

| 方法 | メリット | デメリット |
|------|---------|-----------|
| お名前シール | 貼るだけで簡単。デザイン豊富 | 洗濯で剥がれることがある |
| お名前スタンプ | 布にも押せる。コスパが良い | インクの補充が必要 |
| アイロンシール | 布製品にしっかり定着 | アイロンの手間がかかる |
| 手書き | コストゼロ | 時間がかかる。仕上がりにムラ |

おすすめは「お名前シール＋お名前スタンプ」の併用です。プラスチック製品にはシール、布製品にはスタンプと使い分けると効率的です。

## まとめ

プレ幼稚園は、子どもにとっても親にとっても入園準備の大切なステップです。早めに情報収集を始め、複数の幼稚園を見学して比較することで、お子さんにぴったりの園が見つかるはずです。費用や通園の利便性も重要ですが、何より「子どもが楽しく通えそうか」を基準に選んでください。入園準備グッズも少しずつ揃えて、わくわくの園生活を迎えましょう。
