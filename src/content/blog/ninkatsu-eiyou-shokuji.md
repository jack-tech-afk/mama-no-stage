妊活を始めたとき、「食事で何を意識すればいいの？」と疑問に思う方は多いのではないでしょうか。妊活中の栄養管理は、妊娠しやすい体づくりの土台になります。この記事では、妊活中に特に意識したい栄養素とその食材、1日の食事メニュー例、さらには男性側の食事ポイントや葉酸サプリの比較まで、まるごと解説します。

## 妊活中に特に重要な4つの栄養素

### 1. 葉酸（ようさん）

葉酸は妊活中から妊娠初期にかけて最も重要な栄養素です。赤ちゃんの神経管閉鎖障害のリスクを約70%低減できることが研究で示されています。

| 項目 | 内容 |
|------|------|
| 推奨摂取量 | 食事性葉酸240μg＋サプリで400μg/日 |
| 摂取開始時期 | 妊娠を考え始めたら（妊娠1ヶ月前〜） |
| 多く含む食品 | ほうれん草、ブロッコリー、枝豆、アスパラガス、いちご |
| 注意点 | 食品中の葉酸は吸収率約50%。サプリとの併用が推奨 |

**なぜサプリが必要？** 食事だけでは十分な量を摂取するのが難しいためです。厚生労働省も妊活中の女性にはサプリメントでの葉酸摂取を推奨しています。

### 2. ビタミンD

近年注目が高まっているビタミンDは、妊活との関連が多くの研究で報告されています。

- **子宮内膜の受容能を高める**可能性がある
- **卵巣機能**に影響を与える
- **AMH値**（卵巣予備能の指標）との関連が報告されている
- 日本人女性の**約80%がビタミンD不足**という調査結果も

| 項目 | 内容 |
|------|------|
| 推奨摂取量 | 8.5μg/日（妊活中は25〜50μg/日を推奨する医師も） |
| 多く含む食品 | 鮭、さんま、しらす、きくらげ、卵黄 |
| 効率的な摂取 | 日光浴（1日15〜30分）で体内合成できる |

### 3. 鉄分

鉄分は子宮内膜を厚くし、受精卵が着床しやすい環境をつくるために重要です。

- **ヘム鉄**（動物性） … 吸収率15〜25%。レバー、赤身肉、カツオなど
- **非ヘム鉄**（植物性） … 吸収率2〜5%。小松菜、ひじき、納豆など

**吸収率を上げるコツ**: ビタミンCと一緒に摂ると吸収率がアップします。レモンやブロッコリーを付け合わせにするのがおすすめです。

| 項目 | 内容 |
|------|------|
| 推奨摂取量 | 10.5mg/日（月経ありの場合） |
| 注意点 | コーヒー・紅茶のタンニンは鉄の吸収を妨げるため、食事の前後30分は避ける |

### 4. 亜鉛

亜鉛はホルモンバランスの維持と細胞分裂に関わり、妊活中の男女ともに重要な栄養素です。

- **女性** … 卵子の発育、着床に関与
- **男性** … 精子の形成に不可欠。亜鉛不足は精子の質低下につながる

| 項目 | 内容 |
|------|------|
| 推奨摂取量 | 女性8mg/日、男性11mg/日 |
| 多く含む食品 | 牡蠣（圧倒的1位）、牛肉、豚レバー、チーズ、納豆 |

## 男性不妊と食事の関係

不妊原因の約半数は男性側にあると言われています。男性の精子の質を高めるためにも、食事の見直しは重要です。

### 男性が意識すべき栄養素

| 栄養素 | 精子への影響 | おすすめ食材 |
|-------|------------|------------|
| 亜鉛 | 精子形成に不可欠 | 牡蠣、牛肉、ナッツ |
| ビタミンE | 精子の運動率を改善 | アーモンド、アボカド、かぼちゃ |
| コエンザイムQ10 | 精子の質を向上 | イワシ、牛肉、ブロッコリー |
| DHA・EPA | 精子の膜構造を維持 | サバ、サンマ、イワシ |
| ビタミンC | 精子のDNA損傷を防ぐ | キウイ、パプリカ、レモン |

### 男性が避けるべき食習慣

- **過度なアルコール摂取** … 精子濃度・運動率の低下
- **加工食品の過剰摂取** … トランス脂肪酸が精子の質に影響
- **大豆イソフラボンの過剰摂取** … エストロゲン様作用による影響の可能性
- **カフェインの大量摂取** … 1日300mgまでが目安

## 妊活中の1日食事メニュー例

### 朝食

- 玄米ごはん（150g）
- 鮭の塩焼き（ビタミンD＋タンパク質）
- ほうれん草のおひたし（葉酸＋鉄分）
- 味噌汁（豆腐＋わかめ）
- いちご3〜4個（ビタミンC＋葉酸）

### 昼食

- 牛肉とブロッコリーの炒め物（鉄分＋亜鉛＋葉酸）
- 雑穀米おにぎり
- かぼちゃのサラダ（ビタミンE）
- ミニトマト（ビタミンC）

### 夕食

- カツオのたたき（鉄分＋ビタミンD）
- 枝豆（葉酸＋タンパク質）
- 小松菜と油揚げの煮浸し（鉄分＋カルシウム）
- 味噌汁（なめこ＋豆腐）
- 玄米ごはん

### 間食

- ナッツ類（アーモンド・くるみ）… ビタミンE＋亜鉛
- ヨーグルト＋きなこ … カルシウム＋タンパク質
- キウイ … ビタミンC＋葉酸
