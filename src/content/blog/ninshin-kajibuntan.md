妊娠中の夫婦の悩みで常に上位にランクインするのが**「家事分担」**の問題です。ある調査では、妊娠中のパートナーへの不満第1位が「家事をしてくれない」というデータもあります。体が思うように動かない妊娠中だからこそ、夫婦で家事の分担を見直すことが大切です。この記事では、妊娠の時期に合わせた家事分担表の作り方と、話し合いのテンプレートをご紹介します。

## なぜ妊娠中の家事分担が重要なの？

### 産後クライシスの予防につながる

産後クライシスとは、出産後に夫婦関係が急激に悪化する現象のことです。その原因の多くは、**妊娠中からの家事・育児に対する意識のギャップ**にあります。

妊娠中に家事分担を話し合い、パートナーが家事スキルを身につけておくことは、産後の夫婦関係を良好に保つための重要な「投資」です。

### 妊婦の体への負担を減らす

妊娠中は体にさまざまな変化が起こり、今まで普通にできていた家事が辛くなります。

| 時期 | 体の変化 | 辛くなる家事 |
|------|---------|-----------|
| つわり期（6〜16週） | 吐き気・においに敏感 | 料理、ゴミ出し、トイレ掃除 |
| 安定期（16〜28週） | お腹が出てくる | 床掃除、風呂掃除、布団干し |
| 後期（28〜36週） | お腹がさらに大きく | 前かがみの作業全般 |
| 臨月（36週〜） | 動くのが辛い | ほぼすべての家事 |

## 家事分担表の作り方

### ステップ1: 家事の「見える化」

まずは日常の家事をすべてリストアップしましょう。「名もなき家事」も含めて書き出すことがポイントです。

| カテゴリ | 具体的な家事 |
|---------|-----------|
| 料理 | 献立決め、買い物、調理、配膳、食器洗い、キッチン掃除 |
| 掃除 | 掃除機、床拭き、トイレ掃除、風呂掃除、窓拭き |
| 洗濯 | 洗濯機を回す、干す、取り込む、たたむ、しまう |
| その他 | ゴミ分別・ゴミ出し、買い物、日用品の在庫管理、郵便物の整理、町内会対応 |
| 名もなき家事 | 調味料の補充、トイレットペーパーの交換、排水口の掃除、布巾の交換 |

### ステップ2: 現状の担当割合を把握する

リストアップした家事について、現在の担当者を記入してみましょう。多くの場合、この段階で**妻側に偏っている現実**が可視化されます。

### ステップ3: 時期別に分担を調整する

妊娠の時期ごとに体調が変わるため、固定の分担ではなく**段階的に調整**することが重要です。

#### つわり期（妊娠初期）の分担例

| 家事 | 妻 | 夫 | 備考 |
|------|-----|-----|------|
| 料理 | △ | ◎ | においが辛い時期。夫がメインで担当 |
| 買い物 | × | ◎ | ネットスーパー活用も |
| 掃除機 | △ | ◎ | 週末にまとめて |
| トイレ掃除 | × | ◎ | においが辛いため夫が担当 |
| 洗濯 | ◎ | △ | 比較的できる家事 |
| ゴミ出し | × | ◎ | においが辛いため夫が担当 |

#### 安定期（妊娠中期）の分担例

|------|-----|-----|------|

## 話し合いテンプレート

### 話し合いの前に準備すること

1. **家事リストを作成しておく**（上記のテンプレートを活用）
2. **感情的にならないタイミングを選ぶ**（休日のリラックスした時間）
3. **「責める」ではなく「お願い」のスタンスで**

### 話し合いの流れ

**切り出し方の例**:

「妊娠が進むと体が辛くなってきて、今までどおりの家事が難しくなりそうなの。二人で家事の分担を見直せたらうれしいんだけど、一緒に考えてくれない？」

**具体的に伝えるポイント**:

| NG例 | OK例 |
|------|------|
| 「もっと家事やってよ」 | 「ゴミ出しをお願いできると助かる」 |
| 「何もしてくれない」 | 「お風呂掃除は前かがみが辛いからお願いしたい」 |
| 「普通やるでしょ」 | 「つわりで料理のにおいがきついから、しばらく料理を担当してくれると嬉しい」 |
